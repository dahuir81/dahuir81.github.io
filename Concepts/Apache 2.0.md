# Apache 2.0

**类型**: 开源软件许可证  

## 核心特点

- 允许下载、修改和商业使用
- 只需保留适当的版权声明
- 无自定义条款，无"有害使用"排除条款
- 无再分发或商业部署限制

## 对比 Gemma 旧许可证

| 特性 | 旧 Gemma 许可证 | Apache 2.0 |
|------|-----------------|------------|
| 商业使用 | 有限制 | ✅ 完全允许 |
| 修改分发 | 有限制 | ✅ 完全允许 |
| 法律摩擦 | 高（需合规审查） | 低（标准条款） |
| 社区接受度 | 低 | 高（与Qwen、Mistral一致） |

## 影响

Gemma 4 采用 Apache 2.0 是发布中比 benchmark 数字更重要的变化，消除了法律审查摩擦，使 Gemma 可真正进入生产环境。
