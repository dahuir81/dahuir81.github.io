# DRAM周期

**类型**: 市场周期 / 投资主题
**来源**: [[2026-04-09-UBS-SK-Hynix-Key-Call]]（UBS，2026-04-08）

## UBS判断：30年一遇的存储周期

### 短缺驱动因素
1. **HBM需求蚕食DDR产能** — HBM前端产能占比2026年25% → 2027年31%
2. **传统服务器换机周期**强劲
3. **晶圆产能扩张受限** — 除中国外，新增产能主要投HBM
4. **AI驱动服务器/存储SSD需求**上行

### 结论
**DRAM供应短缺将持续至2027年Q4**

### 价格表现
| 指标 | 2026E YoY | 2027E YoY |
|------|-----------|-----------|
| DRAM ASP | **+167%** | +29% |
| DDR ASP | +264% | +26% |
| HBM ASP | +10% | **+36%** |
| NAND ASP | +213% | +24% |

### 利润率
- DRAM运营利润率：2026E 84.3% → 2027E 86.3%
- SK海力士2026E OP ₩286万亿（超共识57%）

### 与历史周期对比
- 当前ASP涨幅远超以往周期
- HBM带来的结构性变化：产能分配持续向高附加值产品倾斜
- 可能"温和化"传统存储的周期性波动
