## 供需失衡现状

### 需求端爆发

**Agent AI驱动**:
- Claude Code等Agent应用导致token消耗量激增
- SemiAnalysis案例: 7天内消耗数十亿tokens，平均$5/M tokens
- 预计Claude Code将占2026年底全球日代码提交量的20%+

**AI实验室扩张**:
- Anthropic ARR: $90亿 → $300亿+ (单季度3倍增长)
- OpenAI、各类Neolabs融资需求推动GPU采购
- 开源模型(GLM、Kimi K2.5)使用案例激增

**多模态内容生成**:
- Seedance、Nano Banana等视频/图像生成工具
- 用户大规模生成和迭代内容，token吞吐量暴增

### 供应端紧张

**即时容量售罄**:
- 所有GPU类型的按需租赁容量完全售罄
- 锁定按需实例的用户拒绝释放容量回池
- 寻找8节点(64 GPU)的H100/H200都很困难

**合同续约异常**:
- H100以2-3年前价格续约(本应大幅降价)
- 部分合同续约至2028年(4年期)
- 大多数供应商没有Hopper GPU产能即将到期

**Blackwell交付延迟**:
- 新部署交付周期延长至6-7月
- 2026年8-9月前所有新增产能已被预订
- 开放权重模型+推理需求双重挤压

## 涨价驱动因素分析

### 1. AI工具ROI验证完成

**投资回报**:
- 使用AI工具的ROI达到5-10倍
- 成本远低于节省的时间和扩展的工作流能力
- 需求曲线相对无弹性——即使涨价用户仍愿付费

**市场共识形成**:
- "AI工具价值回报"争论已尘埃落定
- 用户从"尝鲜"阶段进入"依赖"阶段
- 当前涨价只是开始，还有很大上涨空间

### 2. 供应链成本螺旋上升

**内存价格暴涨**:
- LPDDR5: 同比上涨约4倍
- DDR5: 同比上涨约5倍
- 基于SemiAnalysis Memory Model数据

**AI服务器重新定价**:
- OEM以显著超过组件成本增幅的水平重新定价
- 压缩预期项目回报，迫使部分运营商放缓部署
- 本应上线的供应被扣留，进一步收紧租赁市场

**硅片短缺**:
- TSMC N3逻辑晶圆产能紧张
- HBM、DRAM、NAND内存短缺
- CPU (AMD、Intel) 同步涨价

### 3. 云厂商策略转变

**全球市场**:
- 从追求利用率转向挑选客户
- 可协商更高预付款、更好定价、更长合同期
- 时间站在Neocloud一边——按自己节奏规划部署

**中国市场**:
- 涨价核心目的: 筛选客户而非转嫁成本
- 淘汰低质量用户、锁定付费客户
- 阿里云取消Lite版，MiniMax等定价更高
- 预判B端Q3爆发，提前定价

## GPU租赁市场结构

### 三大细分市场

```
┌─────────────────────────────────────────────────────────────┐
│                    GPU租赁市场结构                           │
├──────────────┬──────────────┬───────────────────────────────┤
│   短期租赁    │   中期合同    │         长期采购              │
│  (<3个月)    │  (3月-3年)   │        (4-5年)               │
├──────────────┼──────────────┼───────────────────────────────┤
│ • 按需/Spot  │ • 1年期最敏感 │ • AI实验室主导               │
│ • 剩余容量   │ • 边际需求    │ • 50-100MW大型集群           │
│ • 利用率驱动 │ • 1-3年主流   │ • 裸机交易                   │
│ • 价格粘性   │ • AI Native   │ • 20%+预付款                 │
└──────────────┴──────────────┴───────────────────────────────┘
```

### 各细分市场特征

**短期租赁 (On-Demand/Spot)**:
- 代表剩余容量，利用率是需求最佳高频指标
- 价格粘性高，长期持平后突然跳升
- Runpod、Lambda等专注灵活容量

**中期合同 (1-3年)**:
- 1年期捕捉非AI实验室客户的边际需求
- AI Native和中小型AI实验室主要集中于此
- 越来越多实验室寻求4年+长期合同

**长期采购 (4-5年)**:
- AI实验室锁定大量容量(24K-48K GPU)
- 通常涉及Hyperscaler背对背结构
- Neocloud可用合同安排债务融资

## 市场格局重塑

### Neocloud策略180度转变

**从买方市场到卖方市场**:
- 之前: 竞争激烈，多家Neocloud提供低价
- 现在: Neocloud和Hyperscaler主导定价权
- 可挑选合同开始/结束日期匹配库存可用性

**财务影响**:
- 重新加速的租赁定价提升Neocloud ROIC
- 更高租赁价格延长现有GPU经济使用寿命
- 投资资本在需要再投资前产生更长时间现金流

### 云厂商自持算力战略

**大厂转向**:
- 阿里云: 部署大量自研PPU
- 字节跳动: 传出自研芯片+大量国产芯片采购
- 目标: 通过更好PD分离架构降低长期TCO

**火山引擎窗口期**:
- 若其他大厂涨价而火山不跟，是切入大B市场窗口
- 但长期低价不可持续
- 最可能路径: 重点领域保持竞争力，其余领域跟进

### 聚合平台困境

**轻资产模式压力**:
- 硅基流动等自持算力比例<40%
- 大量依赖IDC供应商机房租赁
- 涨价若不能传导到上游，利润空间受挤压

**MaaS平台前景**:
- OpenAI、Anthropic毛利高但可能未真正盈利
- 价格稳定期可通过路由费用存活
- 价格急剧上升环境下空间有限

## 未来展望与监测节点

### 三个关键检查点

**1. GB300集群铺开 (2026全年)**
- 观察新增算力是否缓解计算短缺
- 或token需求是否继续超越供应增长
- 决定AI实验室是否参与<4年市场

**2. 硅片短缺恶化程度**
- TSMC N3产能执行情况
- HBM/DRAM供应链稳定性
- 任何制造环节的执行失误都可能加剧短缺

**3. AI实验室ARR增长**
- 采用率扩散速度
- Token消费持续增长
- SemiAnalysis Tokenomics Model追踪

### 市场共识与现实的脱节

**市场情绪**:
- 对CoreWeave、Nebius、IREN等越来越负面
- 股价处于6-12个月交易区间低端
- 仍锚定于"最终供应过剩和商品化"叙事

**地面现实**:
- 供应持续稀缺
- 所有Neocloud提供商定价权强劲
- 所有计算都有需求，无论相对性能如何

**结论**:
> "This Time *Might* Be Different"
> 
> GPU租赁价格更可能继续上涨而非下跌。动态是自我强化的——供应收紧导致Neocloud提前锁定硬件，进一步收紧供应，推动价格进一步上涨。
