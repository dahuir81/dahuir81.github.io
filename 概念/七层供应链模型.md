# 七层供应链模型

**提出者**: [[Dylan Patel]]  
**来源**: SemiAnalysis  
**核心思想**: 从终端需求倒推瓶颈，而非从芯片向外看

---

## 模型架构

```
Layer 7: 电力
    ↓
Layer 6: 数据中心
    ↓
Layer 5: 网络
    ↓
Layer 4: 服务器
    ↓
Layer 3: 内存
    ↓
Layer 2: 芯片
    ↓
Layer 1: 晶圆
```

---

## 逆向思维

**传统思维**: 芯片 → 服务器 → 数据中心 → 电力
**Patel思维**: 电力 → 数据中心 → ... → 晶圆

**关键洞察**: 从终端需求倒推瓶颈所在

---

## 核心数据

| Layer | 关键数据 |
|-------|---------|
| 7-电力 | 20 GW = 小型国家消耗 |
| 2-芯片 | EUV设备70台，单价$150M |
| 1-晶圆 | 台积电月产能10万片 |

---

## 战略启示

1. **选址策略** > 技术
2. **能源成本**决定AI训练经济性
3. **核能/可再生能源** = 竞争优势
