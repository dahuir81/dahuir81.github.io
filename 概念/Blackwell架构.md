# Blackwell架构

**定义**: NVIDIA于2024年发布的AI加速器架构，用于B200/GB200系列GPU

## 核心规格

| 指标 | B200 | 对比H100 |
|------|------|----------|
| FP16算力 | 2,250 TFLOPS | +127% |
| 显存容量 | 192GB HBM3e | +140% |
| 显存带宽 | 8 TB/s | +67% |
| TDP | 1,000W | +25% |
| 制造成本 | ~$6,400 | +93% |

## 架构创新

### 1. 双芯片设计

- 两个800mm² GPU芯片
- NVLink-C2C互连（10 TB/s）
- CoWoS-L封装

**优势**: 良率优化 + 算力密度提升
**代价**: 封装复杂度 +47%

### 2. 第五代Tensor Core

- 支持FP4/FP6/FP8/FP16/FP32/FP64
- 新增FP4精度（推理加速）

### 3. 第二代Transformer Engine

- 动态精度管理
- 推理吞吐量提升

## 成本结构

```
B200 COGS (~$6,400):
├── HBM3e内存: 45% (~$2,900)
├── 测试组装: 24% (~$1,550)
├── 先进封装: 17% (~$1,100)
└── 逻辑芯片: 13% (~$850)
```
