# Token（AI时代的"度"与"流量"）

**定义**: AI模型处理文本的基本单位，既是计量单位，也是价值载体  
**相关概念**: [[Skill化]]、[[AI替代]]、[[知识蒸馏]]  
**流行语**: "散是Token，聚是skill"

---

## 三重身份

| 应用场景 | Token的含义 | 核心作用 |
|---------|------------|---------|
| **身份验证** | 访问令牌 | 证明"你是你" |
| **大语言模型** | 文本处理最小单位 | AI理解和计费基础 |
| **区块链** | 代币/数字凭证 | 代表数字所有权 |

---

## AI如何"阅读"

### 分词过程
```
"我爱人工智能"
↓
["我", "爱", "人工智能"] ✅
```

### 向量化
- 每个Token → 唯一数字ID
- 通过Embedding → 数千维向量
- 语义相近的词在向量空间中距离近

### 计费模式
- 输入Token + 输出Token
- 按$X/1M tokens计费

### 价值转换
```
人的经验 → Token化 → 训练 → Skill → 替代人
```
