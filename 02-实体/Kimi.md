# Kimi

**类型**: AI 大模型 / 智能助手  
**开发商**: [[月之暗面]]  
**官网**: https://kimi.moonshot.cn

## 核心定位
以超长上下文窗口著称的国产大模型，支持多轮对话、文档理解、网页搜索和代码生成。

## 产品线

### Kimi 智能助手
- 200 万字超长上下文
- 支持文档解析、网页搜索、代码执行

### Kimi Code
- 面向开发者的编程模型订阅服务
- 提供 API 访问，支持不限流调用（相对于按次计费模式）
- 额度按周刷新

### Kimi K2.6
- 月之暗面发布的编程专用模型
- 基于 MoE 架构（1T 参数，激活 32B）
- 被实测认为目前国产编程模型里最强
- 详见: [[Kimi K2.6]]
