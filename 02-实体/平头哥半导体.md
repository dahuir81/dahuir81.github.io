# 平头哥半导体 (T-Head)

## 定义

阿里巴巴旗下半导体公司，成立于2018年，由阿里收购杭州中天微系统后组建。中国领先的AI芯片、存储主控芯片和RISC-V处理器设计公司。

## 核心产品

### 1. 真武PPU (AI芯片)
- **定位**: AI训练/推理加速芯片
- **对标**: 英伟达H20/A100
- **参数**: 96GB HBM2e, 700GB/s片间互联
- **版本演进**: PPU 1.0 → 1.1 → 1.2 → 1.5（快速迭代）
- **交付量**: 47万片（截至2026年2月）
- **年化收入**: 百亿级别
- **客户**: 400+家，60%为外部客户（国家电网、中科院、小鹏等）
- **状态**: 国产GPU出货量第一，供不应求，2025年3月涨价34%

### 2. 镇岳510 (SSD主控)
- **定位**: 企业级SSD主控芯片
- **亮点**: 4μs超低时延，比业界主流低30%+
- **技术**: pSLC+ZNS+QLC组合拳
- **合作**: 忆恒创源、得瑞领新、佰维存储
- **意义**: 补齐阿里AI基建"存力"短板

### 3. 玄铁系列 (RISC-V处理器)
- **C920**: 高性能应用处理器，支持Vector 1.0
- **C907**: AIoT/边缘计算，Matrix扩展
- **生态**: 安卓12、Debian、Ubuntu、OpenHarmony等适配
- **地位**: RISC-V国际基金会12个技术小组主导者

## 竞争格局

### vs 华为昇腾
- 昇腾: 生态完整、政府支持、制裁风险
- 平头哥: 云原生基因、阿里云闭环、商业化更快

### vs 寒武纪
- 寒武纪: 上市早、品牌认知、持续亏损
- 平头哥: 已大规模商用、47万片交付、百亿收入、盈利可期

## 风险因素

1. **地缘政治**: 可能被列入实体清单
2. **人才流失**: 副总裁孟建熠离职
3. **技术差距**: 与英伟达H100/H200仍有代差
4. **生态建设**: RISC-V生态仍处早期
5. **上市不确定**: 独立上市计划未获阿里正式确认
