# Kimi K2.6

**类型**: 大语言模型 / 编程模型  
**开发商**: [[月之暗面]]  
**版本**: K2.6-code-preview (2026-04-13 上线)

## 技术架构
- **架构**: MoE（混合专家模型）
- **参数规模**: 1T 总参数，激活 32B
- **定位**: 编程专用模型

## 核心能力

### 1. 代码能力
- 被实测认为目前国产编程模型里最强
- 思考和执行稳定性超过 GLM 5.1
- 输出结构清晰、前后一致

### 2. 自我纠正
- 在任务跗偏时能主动判断并回滚
- 可自动清理错误产物并重新发起

### 3. 不限流
- Kimi Code 订阅 API 不设速率限制
- 适合多 Agent 并发场景

## 实测痛点

| 痛点 | 说明 |
|------|------|
| 推理速度慢 | 首 token 比 GLM 5.1 慢一个数量级，步骤间明显停顿 |
| 额度消耗快 | 重度 Agent 使用下一天可消耗周额度 24%，频限 78% |

## 适合场景
- **推荐**: 日常编程辅助、需要高质量代码生成的场景
- **谨慎**: 长流程多步 Agent 任务（慢和额度成本会被放大）
