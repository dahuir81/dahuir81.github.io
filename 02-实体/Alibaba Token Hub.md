# Alibaba Token Hub (ATH)

**全称**: Alibaba Token Hub  
**简称**: ATH  
**母公司**: [[02-实体/阿里巴巴|阿里巴巴]]

## 定位

2026年4月正式成立的集团级事业群，负责人为CEO [[06-人物/吴泳铭|吴泳铭]]。

## 业务范围

- **Token生产**: 大模型推理能力输出
- **Token输送**: API与云基础设施
- **Token应用**: 行业解决方案落地

## 战略意义

- 将Token上升为阿里集团级战略，与电商、云智能并列
- 反映了全民"Agent/养虾"热潮下Token消耗激增的市场趋势
