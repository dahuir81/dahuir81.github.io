# 麒麟9030

## 基本信息
- **类型**: 移动SoC芯片
- **制造商**: [[华为]]（设计） / [[中芯国际 SMIC|中芯国际]]（代工）
- **搭载设备**: 华为 Mate 80 Pro Max
- **拆解分析**: TechInsights

## 工艺规格

### 晶体管密度
| 指标 | 数值 |
|------|------|
| 晶体管密度 | 102 MTr/mm² |
| 对比三星/台积电5nm | 不及（<125 MTr/mm²） |
| 对标台积电3nm目标 | 267 MTr/mm² |
| 对标台积电2nm目标 | >300 MTr/mm² |

### 制造工艺
- **代工节点**: 中芯 N+3
- **光刻方式**: DUV多重曝光（无EUV）
- **关键技术**: 自对准四重曝光（SAQP），已超双重曝光极限
- **最小金属间距**: 已使用DUV硬缩至30nm以下

## 技术意义
- 证明了在没有EUV光刻机的情况下，通过DUV多重曝光+合理技术规划，可以达到接近先进节点的密度水平
- 是中国芯片行业在制裁条件下另辟蹊径的重要实证
