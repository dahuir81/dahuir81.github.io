# 套壳之死

**类型**：商业规律 / 行业洞察
**来源**：Sam Altman（"套壳 OpenAI 的公司都会死"）+ Anthropic 死亡更新周期验证

## 定义
建立在别人 API/平台之上的创业公司，其天花板从来不是技术，而是平台方的意愿。当平台方决定自己做同一生态位时，套壳公司的存在前提消失。

##  Anthropic 验证（2026年1-4月）
过去 3 个月，Anthropic 用"死亡更新周期"系统性展示了这一规律：

| 被替代者 | Anthropic 替代功能 | 结果 |
|----------|-------------------|------|
| 法律 SaaS（Westlaw等） | Cowork 法律插件 | Thomson Reuters -16% |
| 金融终端（FactSet等） | Cowork 金融插件 | 初级分析师岗位消失 |
| 安全扫描（Snyk/Veracode） | Claude Code Security | JFrog -25% |
| 自动化平台（Zapier等） | 自动化三部曲 | Zapier 降为可选中间件 |
| AI PPT（Gamma/Tome） | PowerPoint 原生集成 | 独立平台失去存在理由 |
| **OpenClaw** | Claude Code Channels + 订阅封锁 | 必须改用 API，成本涨数十倍 |
| AI Infra（LangChain/Dify） | Managed Agents | Cloudflare 三个月被冲击三次 |

## 核心机制
不是"产品更好竞争赢了"，而是**定价逻辑失去活路**：
- 按人头卖账号的 SaaS → AI 让人均产出×10 → 账号需求萎缩
- 规则库作为护城河 → 语义理解不需要规则库 → 壁垒变转型包袱
- 中间件作为必选项 → 平台原生集成 → 降级为可选层

## 深层逻辑
**建立在别人 API 上构建商业模式，天花板从来不是技术，而是平台方的意愿**。
