## 核心事件
Anthropic 过去 3 个月平均每两周一次重量级发布，系统性地向应用层扩张。每次发布都有一批公司的存在前提消失。

## 死亡名单

| 时间 | 发布 | 冲击对象 |
|------|------|----------|
| 1月30日 | Cowork 法律/金融/营销插件三连发 | Thomson Reuters (-16%)、LegalZoom (-20%)、FactSet、S&P Global |
| 2月20日 | Claude Code Security | Veracode、Snyk、JFrog (-25%)、GitLab (-8%) |
| 2月25日-3月26日 | 自动化三部曲 | Zapier、Make.com、n8n、UiPath |
| 3月11日 | PowerPoint 原生集成 | Gamma、Tome、Beautiful.ai |
| 3月24日 | Claude Code Channels | OpenClaw（IM远程操控核心功能被抄） |
| 4月4日 | 切断第三方订阅凭证 | OpenClaw等框架必须改用API，成本涨数十倍 |
| 4月7日 | Claude Mythos（Project Glasswing） | CrowdStrike、Okta、Zscaler、Cloudflare |
| 4月8日 | Managed Agents 公测 | LangChain、StackAI、Dify、Cloudflare (-13.5%) |

## 两种武器
1. **产品能力**：把对手的功能做成自己的功能
2. **平台权力**：修改规则，切断第三方框架的订阅访问权限

## 对OpenClaw的影响
- Claude Code Channels 直接抄走 IM 远程操控核心功能
- 4月4日条款更新，切断第三方通过订阅凭证访问
- 必须改用按量计费 API，成本涨数十倍
- 在 Anthropic API 上构建商业模式，天花板不是技术，是 Anthropic 的意愿

## 商业信号
- Lovable 估值 66 亿美元，Claude 即将推出类似功能
- Cloudflare 三个月内被冲击三次
- AI 让人均产出×10，按人头卖账号的 SaaS 模式崩塌
- "套壳 OpenAI 的公司都会死" → 现在 Anthropic 展示了同一逻辑
