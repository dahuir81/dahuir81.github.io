# 阿里确认HappyHorse为ATH创新事业部研发，郑波团队负责

> 来源：APPSO（爱范儿旗下）
> 作者：未署名

> 抓取时间：2026-04-10 14:25

## 核心信息

### 官方确认
- HappyHorse 为阿里 ATH 创新事业部研发
- 通过官方微博 @HappyHorse_AI 正式确认
- 目前处于内测中，近期开放 API

### 负责人：郑波
- 阿里巴巴副总裁
- 清华大学计算机系博士
- 2006-2017：Google 展示广告算法团队负责人 + 中国地图团队负责人
- 2017.09 加入阿里：淘搜推算法负责人 → 阿里妈妈CTO → 淘天集团算法技术负责人
- 研究方向：大模型、多模态、决策智能、深度学习、搜推广算法

### 人工析排名确认
- 无音频排行榜：第一
- 有音频排行榜：Elo 分数与 Seedance 2.0 几乎相同
- 支持四种模式：文转视频（有/无音频）、图转视频（有/无音频）
- API 计划：4月30日开放

### 组织架构背景
- 3月16日：成立 Alibaba Token Hub "ATH" 事业群，吴泳铭直接负责
- 4月8日：成立集团技术委员会，通义实验室升级为通义大模型事业部
- 23天内完成两次 AI 组织架构调整

### 纠正早期报道
早期报道（The Information 等）将 HappyHorse 归为张迪（前快手副总裁）团队，但阿里官方确认为郑波团队。张迪的具体角色待确认。
