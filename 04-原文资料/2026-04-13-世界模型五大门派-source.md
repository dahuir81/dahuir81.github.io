## 背景

- 杨立昆的 AMI 拿了 **10.3 亿美元**融资，创下欧洲 AI 公司种子轮纪录
- 李飞飞的 World Labs 宣布 **10 亿美元**融资
- 两家公司，同一赛道：**世界模型（World Model）**
- 世界模型成为科研精英接下来攻关方向的原因：LLM 训练数据是文本，真实世界是三维的、连续的、充满物理规律的

> 杨立昆断言：五年后没有人会再用今天这样的大语言模型
> AI 已经读完了万卷书，接下来该行千里路了。世界模型会成为它的眼镜和四肢。

## 五大门派

### 1. JEPA 派（杨立昆·抽象预测）

| 项目 | 内容 |
|------|------|
| 全称 | Joint-Embedding Predictive Architecture（联合嵌入预测架构） |
| 核心思想 | 让 AI 通过"观察"学习世界运行规律，而非死记硬背像素 |
| 关键区别 | 不预测像素，在抽象的"表征空间"（latent space）里预测"会发生什么" |
| 代表作 | V-JEPA 2（12 亿参数，100 万小时无标签视频预训练） |
| 数据效率 | 仅需 62 小时机器人数据，零样本规划动作，成功率 65-80% |
| 商业化 | AMI 公司，可能几年后才能看到产品，长线投资 |
| 投资方 | 几乎所有业界和跨界大佬 |

**杨立昆原话**：如果表征足够好，你就不需要为每个任务从头训练。

### 2. 空间智能派（李飞飞·建筑师）

| 项目 | 内容 |
|------|------|
| 公司 | World Labs |
| 核心思想 | 对三维世界有显式理解：几何结构、深度关系、物体相对位置 |
| 产品 | **Marble**（2025 年 11 月发布）——输入文字/照片/视频/3D 草图，输出可编辑 3D 世界 |
| 技术特点 | 空间一致性，底层维护真实空间表示 |
| 团队 | 联创 Ben Mildenhall（NeRF 发明者）、Christoph Lassner（3D 图形专家） |
| 融资 | 2026 年 2 月完成 10 亿美元，投资方 NVIDIA/AMD/Autodesk |
| 局限 | 生成 3D 世界走几步后出现视觉变形（"幻觉"）；对"会发生什么"理解薄弱 |

**与李飞飞 JEPA 的区别**：
- JEPA 教 AI 理解"球会从桌子上滚下去"（事件逻辑链条）
- World Labs 教 AI 理解"桌子有多高、球在什么位置"（空间物理结构）

### 3. 学习型仿真派（DeepMind·造梦师）

| 项目 | 内容 |
|------|------|
| 代表 | Genie 3（环境生成）+ DreamerV4（虚拟训练） |
| Genie 3 能力 | 输入一句话生成 720p 24fps 3D 环境，支持实时交互、对象持久性 |
| DreamerV4 成就 | 第一个纯靠离线数据在 Minecraft 挖到钻石的 AI，数据量只有 OpenAI VPT 的 1% |
| 局限 | 只能维持几分钟连贯性；像素生成≠物理理解；没有硬编码物理引擎 |

### 4. 基础设施派（英伟达·卖水卖铲子）

| 项目 | 内容 |
|------|------|
| 平台 | Cosmos（2025 年 1 月 CES 发布） |
| 定位 | 为所有路线提供基础设施 |
| 核心组件 | Cosmos Curator（14 天处理 2000 万小时视频）、视觉 Tokenizer（压缩率高 8 倍）、预训练模型家族 |
| 预训练模型 | Cosmos Predict（预测环境未来）、Cosmos Transfer（仿真迁移到真实）、Cosmos Reason（机器人规划） |
| 客户 | 小鹏（自动驾驶仿真）、Figure AI（机器人训练数据） |
| 商业模式 | 免费开放软件锁定硬件生态（CUDA、H 系列 GPU、Jetson） |

> 黄仁勋原话：世界基础模型之于物理 AI，就像大语言模型之于生成式 AI。

### 5. 主动推断派（Verses·神经科学异端）

| 项目 | 内容 |
|------|------|
| 理论基础 | 卡尔·弗里斯顿（Karl Friston）的"自由能原理" |
| 核心逻辑 | 不是"奖励最大化"，而是"减少意外"——不断生成预测，采取行动减少偏差 |
| 框架 | AXIOM |
| 特点 | 对象中心化（离散对象+属性+关系）、贝叶斯推理（概率分布+消息传递，非梯度下降） |
| 不需要预训练 | 面对陌生物体实时重新规划，在线推理而非执行固定策略 |
| 成绩 | 雅达利 GameWorld 10K 基准超越 DreamerV3，数据量远少，步数只有 1/8 |
| 商业化 | Genius 产品，面向金融/机器人/智慧城市 |
| Gartner 研报 | 被写入 AI、空间 AI、物理 AI 报告 |

**比喻**：
- 强化学习 = 被胡萝卜引导的驴子（逐利，目标驱动）
- 主动推断 = 陌生城市的旅行者（避险，好奇心驱动）

## 对比总结

| 门派 | 代表 | 核心优势 | 主要挑战 |
|------|------|---------|---------|
| JEPA | AMI | 数据效率极高，抽象表征 | 商业化周期长 |
| 空间智能 | World Labs | 可编辑 3D，设计/游戏刚需 | 物理动态性较弱 |
| 学习型仿真 | DeepMind | 交互性强，训练成本低 | 像素生成≠物理理解 |
| 基础设施 | 英伟达 | 全栈平台，生态锁定 | 依赖硬件销售 |
| 主动推断 | Verses | 实时适应，可解释性强 | 生态兼容性差 |

## 核心观点

1. 五大门派不是非此即彼，更像是同一块拼图的不同碎片
2. 大语言模型边际收益递减，幻觉、推理能力、多模态理解仍是硬伤
3. 具身智能崛起：人形机器人、自动驾驶、工业自动化需要 AI 与物理世界交互
4. 纯文本训练的模型解决不了物理交互问题
5. 长期来看，**融合是唯一出路**——真正的智能大概不会只有一种形态
