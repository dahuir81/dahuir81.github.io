2026 算力市场全景透视

运营商 VS 云厂商 VS 中转站——谁在定义 AI 时代的"石油"定价权

#[ 2026-05-12 ] #[ 深度分析 ] #[ 算力市场 ]

[引言] THESIS

2026 年春天，算力市场发生了一件微妙的事：Token 从技术指标变成了商品标签。中国移动和中国电信在同一个月内发布了明确的 Token 战略；傅盛推出 Easy Router 入局 API 中转，孙宇晨的 B.AI 用户破百万，特朗普家族推出 WorldClaw 聚合 300+ 模型；与此同时，DeepSeek V4 Flash 把推理成本打到了 $0.14/百万 token，小米 MiMo 跟进，云厂商集体推出 Coding Plan 套餐。

三股力量同时涌向 Token 市场——运营商、API 中转站、云厂与模型厂——各自的打法、宿命和终局完全不同。本文试图拼出 2026 年算力市场的完整图景，并推演未来两到三年的走向。

三路资本同时在赌同一件事——Token 是 AI 时代的石油。但通往定价权的路，至少有三条。

[01] 运营商内战：同一张 Token 牌，两种完全不同的打法

2026 年一季度，三大运营商净利润全线下滑——中国电信 -17.08%、中国联通 -18%、中国移动 -4.2%。传统通信业务失速已成不可逆趋势。Token 经营，是这三家公司共同的救命稻草。但战略路径的分歧远超想象。

TELECOM 中国电信：服务打天下

核心逻辑：Token 只是新的计量单位，客户买的是 AI 服务。

关键布局：星辰 TokenHub（模型聚合路由）、AI STORE（智能体商店）、天翼智铃/智屏/智看（硬件入口）。

核心资产：3.7 万家行业客户、80+ 央企合作、FDE 前沿部署工程师队伍驻扎客户现场。

盈利模式：端到端行业解决方案，卖的是服务而非 API 调用量。将 Token 嵌入月费，追求高毛利项目型收入。

MOBILE 中国移动：基础设施为王

核心逻辑：搭好算网底座和模型平台，让应用在上面跑，赚 Token 调度费。

关键布局：算网大脑（百万亿级 Token 调度、800GB 超高吞吐）、MoMA 平台（300+ 模型聚合）、MobileClaw 智能体框架（150+ 行业 Skills）。

关键武器：AI-eSIM——将智能大脑植入 SIM 卡芯片，终端每次自主思考都在消耗移动的 Token。

盈利模式：规模化调度，算电协同降成本，边际成本随规模递减。

1

两家的模型聚合，本质都是 API 中转站

电信的星辰 TokenHub 和移动的 MoMA，都在做同一类生意：模型聚合。往前半步就是 API 中转站。这个第一层价值——帮客户降低选择成本和集成成本——本质上是信息差的辛苦钱，护城河很浅。

2

移动的抬升路径：算力网 + AI-eSIM

纯中转站只能做云端分发。移动的算网大脑能在全国范围内调度 Token——在正确的时间、以正确的成本和延迟、路由到正确的算力节点。AI-eSIM 把入口焊死在终端硬件层，每一次自主思考都在计费。两端的夹击，把中转站做成了基础设施。

3

电信的抬升路径：服务层加厚

星辰 TokenHub 只做聚合分发利润薄。但把聚合来的模型连同智能体、行业数据集、安全合规方案、FDE 部署服务打包，做成行业解决方案交付出给央企和政企客户——这时候卖的不是 API 调用量，是端到端服务能力，毛利率完全不同。

电信的模式短期收入确定性更强；移动的模式长期边际成本更低。两家的共同软肋是：Token 如果始终停留在客户无感知的底层计量单位，那 Token 经营就永远停在 PPT 上。

[02] 中转站狂欢：傅盛、孙宇晨、特朗普，暴利与灰产

2026 年，API 中转站成了 AI 圈最暴力的造富故事。傅盛的 Easy Router 打出 DeepSeek-V4 定价低至官方四分之一的招牌；孙宇晨的 B.AI 宣称突破 100 万用户；特朗普家族的 WorldClaw 项目聚合 300+ AI 模型。灰色地带的代购生意，突然站到了聚光灯下。

TIER 1 正规军：赚辛苦钱

代表：OpenRouter（300+ 模型，500 万+用户，月处理 70 万亿 token），傅盛的 Easy Router。

盈利方式：充值时收 5.5% 手续费，不在 Token 单价上溢价。

毛利率：5-15%。靠上市公司主体公信力做合规中转，微利但走得稳。

TIER 2 暴利区：号池 + 超售

运作方式：批量注册海外账号，薅免费额度；Sub2API 把网页端逆向成标准 API。

暴利公式：一个 Claude Code Max 账号（$200/月）切给 20 人用，产出 $2000-5000 收入。10-25 倍。

毛利率可达 70%。月流水 500 万的头部玩家很常见。

TIER 3 黑产线：无底线

模型掉包：声称 Claude Opus，后台跑开源小模型，成本差几十倍。

Token 虚标：实际消耗 100 个 Token，后台记 200-300 个。

"三头吃"：收充值费 + 白嫖上游额度 + 卖对话数据。学术论文发现 45.83% 的影子 API 节点通不过模型身份验证。

DATA 安全黑洞

CISPA 审计发现：428 个被测中转站中，9 个向用户注入恶意代码，17 个触发 AWS 密钥盗用，1 个直接抽走私钥钱包资金。

多跳路由链条往往经过 4 个以上节点，最弱的环节决定安全性。LiteLLM 事件曾波及 4.6 万个开发环境。

当国产大模型价格被打到海外厂商的几十分之一，当 DeepSeek V4 Flash 的百万 token 输出只要 0.28 美元——拧开水龙头就能喝到便宜的水时，暗巷里倒卖高价水的生意必然萎缩。

[03] 云厂与模型厂：一场没有底线的价格战

2026 年 5 月，全球 AI API 价格地图已经面目全非。旗舰模型的价格在 $5-30/百万 token 区间，但预算型模型已杀到 $0.10-0.30。中国市场尤为惨烈。

[04] 算力市场未来走向推演

A

中转站的宿命：潮水退去，裸泳者现

纯 API 中转站的毛利会越来越薄，这是确定性很高的事。Anthropic 加速收紧政策（4 月 5 日宣布 Claude 订阅不再支持第三方 harness 平台），监管端《生成式人工智能服务管理暂行办法》持续施压。国产模型价格已触底，信息差和渠道差正在消失。第一档（正规企业级）还能靠合规和服务生存，第二档和第三档面临持续缩窄的套利空间。

B

运营商的胜负手：算力调度 vs 服务厚度

移动的算网大脑和 AI-eSIM 提供了从"管道"升级为"平台"的可能性——如果 Token 能在移动的算力网内闭环流转，且 AI-eSIM 成为行业标配，移动就拿到了智能终端的"每一次思考"的计费权。电信的 FDE 队伍和政企关系网提供了项目制高毛利的确定性，但天花板取决于能不能跑出爆款 AI 产品。

C

云厂商的终极优势：模型层自有 + 生态锁定

阿里云百炼（千问）、火山引擎（豆包）、腾讯云（混元）——云厂商在模型层完全自有，每笔 Token 的收入 100% 归自己。叠加各自的 AI 入口：Coze 开发者平台（100 万+活跃开发者）、通义 App（MAU 破亿）、微信智能体（小程序+支付闭环）。运营商"云、网、边"的基础设施优势，正在被云厂商的"模型、平台、生态"组合拳抵消。

D

定价权的三条路径

路径一：Token 成为标准化的底层计量单位，算力像电力一样按度计费。运营商的基础设施优势最大化，但利润率被压缩到公共事业级别。
路径二：Token 始终封装在应用和服务中，客户只感知结果。云厂商和模型厂通过生态锁定获取超额利润，运营商被挤回管道角色。
路径三：AI-eSIM 和端侧智能让运营商重新获得终端入口，移动的芯片级嵌入策略开辟新战场——但这需要 eSIM 硬件生态在机器人、智能汽车、AIoT 领域的渗透率达到临界点。

2026 年的算力市场，三路资本同时在赌 Token 是 AI 时代的石油。但石油时代的赢家不是采油厂，是标准制定者和基础设施拥有者。谁能在 Token 的世界里成为那个制定规则的人，谁才算拿到了未来的船票。
