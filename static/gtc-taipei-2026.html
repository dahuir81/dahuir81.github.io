"四十年后，英伟达要和微软，一起重新发明 PC。大模型就是新时代的 DirectX，Agent 就是新时代的应用。" —— Jensen Huang

关键护城河：RTX Spark 第一次在同一颗芯片上实现了 统一内存 + 完整 CUDA 生态。在此之前这两件事在 PC 上不可兼得——Mac 有统一内存但没 CUDA，传统 N 卡有 CUDA 但被显存容量卡死。128GB 统一内存意味着可以本地流畅跑 70B–120B 参数模型，还能做 LoRA 微调、RAG、Agent 自迭代。

合作伙伴：联想、HP、联发科、微软（Windows 原生 Agent 重构）、Adobe（2 倍加速 + 原生 Agent 调用）

Cosmos 3 — 全球首个全开源全模态物理 AI 模型

基于突破性 Mixture-of-Transformers 架构，Cosmos 3 是一个 omnimodel（全模态模型），能原生理解和生成文本、图像、视频、环境声音和动作数据。之前 Cosmos 需要多个模型拼凑，现在一个模型搞定。

世界生成

数月→数天

完全开源

关键能力：视觉推理 → 生成物理场景 → 输出关节角度、夹爪位置、轨迹点等数值动作数据，让机器人直接执行。同时成立 NVIDIA Cosmos 联盟，成员包括思灵机器人、Runway、LTX、Skild AI 等。

客户/合作方：思灵机器人、斗山机器人、LG、三星、Skild AI、理想汽车、Linker Vision、Agile Robots

开源大模型

Nemotron 3 Ultra — 英伟达最大开源模型

约 5500 亿总参数，每 token 激活 550 亿参数。英伟达迄今最大的开源模型，定位为 Agent 基座模型。

开源模型

Alpamayo 2 Super — 机器人出租车推理模型

320 亿参数（上一代 100 亿的 3 倍），基于 Cosmos 世界基础模型。从单目前视扩展到 360° 全环绕感知，增加 Meta-Action 输出。配套 AlpaGym（闭环强化学习框架）、OmniDreams（生成式世界模型用于长尾场景模拟）、Omniverse NuRec（神经重建）。

Agent 平台

NVIDIA Agent Toolkit — 四层 Agent 系统

老黄在这次 GTC 上拿出了完整的 Agent 基础设施，四层架构：

模型层Nemotron 3 Ultra + Cosmos 3
框架层Claude Code / Codex / OpenClaw 等直接运行
工具层CUDA-X 全部附带 skills 文件，Agent 自学即用
运行时层OpenShell（开源安全运行时，Apache 2.0）

同时发布开源 NVIDIA OpenShell 运行时，提供基于策略的安全与隐私治理，支持本地或云端部署。

英伟达第一次拿出完整的人形机器人方案，把 "身体 + 大脑 + 软件" 整合进一个开放平台。由宇树科技在 2026 年底生产。

身高约 1.8 米
重量约 68 公斤
自由度75 DOF（身体+手）
手臂扭矩120 Nm
腿部扭矩360 Nm

核心配置：宇树 H2 Plus 本体 + Sharpa Wave 五指触觉灵巧手 + NVIDIA Jetson AGX Thor T5000（Blackwell GPU / 2070 FP4 TFLOPS / 128GB 统一内存）+ Isaac GR00T 开源软件。研究机构包括 Ai2、ETH Zurich、Stanford Robotics Center、UC San Diego 已确认采用。

产业生态与基础设施

Ecosystem
AI 工厂

DSX — 从卖芯片到卖整座 AI 工厂

英伟达推出 DSX（Data Center eXperience）——一套 AI 工厂的完整施工蓝图。在 Omniverse 里建数字孪生模拟电力/散热/网络，通电后 DSX OS 接管运营。DSX Max LPS 解决 AI 工厂普遍超配电力 40% 的痛点。

"算力就是营收，算力就是利润。没有营收和利润，就是亏损。买越多，赚越多。" —— Jensen Huang

老黄预言：到本世纪末，100 吉瓦的 AI 工厂将陆续上线。

智能工厂

Factory Operations Blueprint (FOX) — 工厂管理 Agent

参考设计，用于构建自主工厂管理 Agent，持续监控实时数据并协调专业 Agent 团队。基于 NemoClaw、AI-Q Blueprint 和 Nemotron 模型。DGX Station 工作站级运行。

Foxconn 落地效果：根因分析提速 80%、劳动生产效率提升 15%、设备故障率降低 10%。Foxconn 也在用 DeepHow 视觉 AI + Cosmos 提升一次合格率 3%。

生态扩展

AI Cloud Ecosystem — 六大洲覆盖

NVIDIA AI Cloud 生态加速全球 AI 工厂基础设施建设，覆盖 CoreWeave、Lambda、Nscale 等。新增非洲 Cassava 和南美 Claro，首次实现六大洲覆盖。

半导体

TSMC × NVIDIA — AI 进入晶圆厂

台积电将 CUDA-X 库和 AI 模型应用于计算光刻（cuLitho 提升 20-50%）、晶体管仿真（cuEST 提速 50×）、先进制程控制、良率分析和缺陷检测。

医疗

Healthy Taiwan — Foxconn + 台湾医疗中心

Foxconn 和台湾顶尖医疗中心部署 NVIDIA 驱动的专用 Agent 团队，推动 AI 驱动的健康系统转型。
