🎙️ 黄仁勋 GTC 2026 六大核心论点
🏭 1. Token Factory 框架

数据中心被重新定义为「生产 Token 的工厂」。电力 = 产能上限，Token/s = 产量，$/M tokens = 单件成本。衡量 AI 基建 ROI 不再看 TFLOPS，看每瓦产出多少 Token。

💰 2. 推理取代训练成为经济引擎

「训练是一次性投资，推理是持续的现金流业务。」每个用户每次提问都是新的推理请求。Agentic AI 的爆发进一步放大了推理需求——每个 Agent 需要持续对话、检索记忆、实时响应，Token 消耗量成倍增长。

🔧 3. 极端协同设计（Extreme Co-Design）

每一层都在榨效率：

cuLitho 光刻加速 70x cuEST 量子化学材料模拟 HBM4 内存带宽 22 TB/s NVLink 6 互联 100% 液冷 → PUE 1.1 Dynamo/FM 全栈调度
📏 4. Intelligence per Watt 成为核心投资指标

未来基建投资人衡量项目的第一个指标不是「几兆瓦」，而是「你这兆瓦能产出多少 Token」——这改变了数据中心估值逻辑。

🔥 5. 单 GPU 功耗 1800-2300W，100% 液冷强制

Rubin GPU 功耗是 H100 的 4-5 倍。传统风冷机架设计彻底失效，数据中心面临全盘冷却架构重做。功率密度翻倍意味着传统 1MW 机柜只能放 400-500 块 Rubin GPU。

🌍 6. 全球 AI 基建投资 → $2 万亿

黄仁勋预测未来 4-5 年全球 AI 工厂投资将达到 2 万亿美元。「每个工程师/开发者每年应该至少花 $250K 在 Token 上——否则我会发飙。」
