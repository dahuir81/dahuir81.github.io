# Palantir

**类型**: 公司  
**领域**: AI 数据平台、企业软件  
**成立时间**: 2003年（23年历史）

## 简介
Palantir 是全球领先的 AI 数据平台公司，以 Foundry 平台为核心，提供 Context + Harness 架构的 "AI 操作系统"。

## 财务表现（2025 Q4）

| 指标 | 数据 |
|------|------|
| Q4 营收 | 14亿美元，同比 +70% |
| 美国企业业务 | 暴增 +93% |
| 利润率 | 57% |
| 市值 | 超过 SAP 的 1.7 倍 |
| 股价涨幅（2023至今） | 20倍+ |

## 核心产品

### Foundry
- **定位**: AI 操作系统
- **架构**: Context + Harness
- **功能**: 企业数据治理、AI 应用编排、规则校验

## 竞争优势
- **23 年数据治理经验**: 帮军队、跨国巨头清洗底层数据
- **企业级上下文**: 掌握 "我的公司" 的数据、权限、流程
- **大模型补齐拼图**: 提供交互入口，让复杂系统变得易用

## 商业模式
- 不卖模型，卖 "AI 基础设施"
- 客户接入 Foundry，AI 即可接管业务
