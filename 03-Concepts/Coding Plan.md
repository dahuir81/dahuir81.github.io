# Coding Plan

**类型**: 商业模式/订阅服务

## 定义
Coding Plan（编程套餐）是大模型厂商针对 OpenClaw 等 AI 编程工具推出的按月订阅服务，用户支付固定月费即可获得大量 Token 额度，不再按 Token 单独计费。

## 背景
- 传统模式: B 端卖 API，C 端付费模式被「锁死」
- OpenClaw 出现: 单次任务消耗百万级 Tokens，按量付费可能烧几十块
- 转变: 按月订阅变得划算，Coding Plan 从「提前布局」变成「刚需订阅」

## 智谱激进策略
- 体验月卡: 39元/3500万 Tokens
- 进阶月卡: 99元/1亿 Tokens
- 从 API 售卖转为「数字员工」订阅服务
- 2026 Q1: API 调用定价提升 83%，国内付费 Token 消耗量最高

## 行业数据
- 月之暗面: 2月20天内收入超过2025年全年总额，海外收入占比超70%
- MiniMax: 2026年2月编程套餐 Token 消耗量是2025年12月的10倍以上

## 核心问题: 续费难题

### 尝鲜型消费
- 大量用户冲着限免和首月优惠来
- 真正持续付费的核心用户远低于预期

### 三大障碍
1. **配置门槛** - 需集成飞书、Notion、GitHub 等工具
2. **任务稳定性** - 超过10步的任务易幻觉、中断
3. **必用场景** - 没有哪件事是非 OpenClaw 不可的

## 与海外工具对比
| 维度 | Cursor/Copilot | OpenClaw |
|------|---------------|----------|
| 嵌入工作流 | 直接嵌入开发者日常 | 通用助理 |
| 付费率 | 高 | 首月高，续费低 |
| 用户粘性 | 每天必用 | 尝鲜后流失 |
