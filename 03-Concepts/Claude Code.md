# Claude Code

**类型**: AI编程工具  
**开发商**: Anthropic

## 简介
Claude Code是Anthropic推出的AI编程助手，集成在终端中，帮助开发者更高效地编写、理解和调试代码。

## 核心功能
- 代码生成与补全
- 代码审查与优化建议
- 项目理解与导航
- 自然语言交互编程
