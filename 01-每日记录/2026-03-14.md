## Brave API Key 配置问题

**问题**: web_search 工具多次配置 Brave API Key 后丢失，无法持久化。

**根本原因分析**:
- 3月14日 00:02 之前 web_search 正常工作
- 3月14日 08:02 开始报错 `missing_brave_api_key`
- 检查发现 `~/.openclaw/openclaw.json` 中从未持久化 `tools.web.brave.apiKey` 配置
- 之前可能通过环境变量 `BRAVE_API_KEY` 临时配置，Gateway 重启后丢失

**解决方案**:
- 需要将 Brave API Key 写入 `~/.openclaw/openclaw.json` 文件
- 配置路径: `tools.web.brave.apiKey`
- 避免使用环境变量，确保配置持久化

**相关文件**:
- `~/.openclaw/openclaw.json` - 主配置文件
- `~/.openclaw/backups/` - 配置备份目录
