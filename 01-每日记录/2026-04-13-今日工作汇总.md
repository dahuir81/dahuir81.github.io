### 下午 14:07 — 安全事件：QQ 邮箱 SMTP 授权码泄漏

**事件**：QQ 邮箱（5283728@qq.com）从 06:47 起被攻击者利用，群发 227 封法语钓鱼邮件。

**根因**：`blog/notify.sh` 中硬编码 SMTP 授权码，通过博客发布流程推送到公开 GitHub 仓库，被自动化爬虫抓取。攻击源 IP：98.81.133.132（美国弗吉尼亚）。

**处置**：
1. 停用全部 4 个授权码
2. `git filter-branch` 从 128 个提交历史中清除 `notify.sh`
3. 创建 `.gitignore` 排除敏感文件
4. 迁移邮件通知至 GitHub Secrets 机制
5. 重新生成授权码

**教训**：绝不在公开仓库硬编码任何密码/Token；泄露后立即全量停用而非逐个排查。
