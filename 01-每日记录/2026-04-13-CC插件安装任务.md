# Claude Code 插件安装任务

> 来源：慧哥分享的微信文章，3 个 Claude Code 开源项目
> 环境：macOS M1，Claude Code 已安装
> 要求：按顺序执行，每步安装后验证，最后给一份简短报告

---

## 插件 1：andrej-karpathy-skills

**作者**：Karpathy 的编码陷阱指令集  
**仓库**：https://github.com/forrestchang/andrej-karpathy-skills  
**用途**：把 Karpathy 观察到的 LLM 编码常见陷阱写入 CLAUDE.md，让 CC 在执行编码任务时自动规避

### 安装步骤

```bash
cd ~
git clone https://github.com/forrestchang/andrej-karpathy-skills.git
cp andrej-karpathy-skills/CLAUDE.md ~/.claude/
```

### 验证

```bash
cat ~/.claude/CLAUDE.md | head -20
ls -la ~/.claude/CLAUDE.md
```

确认文件存在且内容非空即可。

## 插件 2：claude-code-best-practice

**仓库**：https://github.com/shanraisshan/claude-code-best-practice  
**用途**：社区整理的 Claude Code 最佳实践集合，作为参考文档使用

### 安装步骤

```bash
cd ~
git clone https://github.com/shanraisshan/claude-code-best-practice.git
```

### 验证

```bash
ls claude-code-best-practice/
cat claude-code-best-practice/README.md | head -30
```

确认 README 能正常阅读。

---

## 插件 3：claude-mem

**仓库**：https://github.com/thedotmack/claude-mem  
**用途**：跨会话记忆，自动捕获和存储 Claude 编码会话上下文，支持多次对话间共享上下文

### 安装步骤

```bash
cd ~
git clone https://github.com/thedotmack/claude-mem.git
cd claude-mem
npm install
```

### 验证

```bash
ls -la node_modules/ | head -5
cat package.json | grep -A 5 '"scripts"'
```

按 README 中的配置说明完成配置，确认能正常工作。
