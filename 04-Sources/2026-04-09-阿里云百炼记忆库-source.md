# 阿里云百炼记忆库上线

今天，阿里云百炼正式上线「记忆库」功能，让Agent具备跨会话的长期记忆能力，真正实现「越聊越懂用户」的个性化体验。

现在，「记忆库」功能限时免费向所有用户开放，用户可通过API直接调用，或通过OpenClaw等Agent产品一键安装。

## 核心架构

阿里云百炼推出的「记忆库系统」内置了「提取-存储-检索-注入」四大模块，用户每次与AI Agent对话结束后，系统可根据配置的记忆规则自动提取关键信息并存储，并根据用户的指令，触发语义检索召回相关记忆并附加至上下文中，实现个性化回答，有效提升Agent的长期记忆能力。

### 开箱即用的记忆策略

Agent开发者可通过白盒化的配置方式，快速配置记忆规则，包括用户的画像，如"性格"、"消费习惯"，也可配置记忆片段抽取规则，如"关注用户的家庭关系情况"。

同时记忆库内置多种通用规则，满足消费电子、客服销售、情感陪伴等多场景需求，配置成本下降50%。

### 动态记忆治理机制

记忆库可自动从用户上下文交互中自动识别并提取记忆，同时动态跟踪变化，自动更新记忆内容，过滤记忆冲突，自主记忆遗忘，赋能Agent实现更「类人」的个性化体验，模拟人类记忆曲线，对低频、过时信息进行衰减或归档，防止上下文污染，确保检索结果的高信噪比。

### 高性能记忆检索引擎

记忆库在检索上基于特殊算法优化，在检索性能、日程类回答以及内容相关性上提供更强的方案，实现记忆搜索判定平均RT下降50%，日期相关性提升66%，记忆内容相关性提升39%，让Agent的每次记忆检索更高效更精准。

## 3分钟让OpenClaw拥有专属永久记忆实操教程

### 步骤一：安装记忆插件（一行命令）

```bash
openclaw plugins install @modelstudio/modelstudio-memory-for-openclaw
```

### 步骤二：获取API Key

登录阿里云百炼控制台，进入「API Key 管理」页面，创建或复制现有Key。

密钥获取地址：https://bailian.console.aliyun.com/cn-beijing?tab=app#/api-key

### 步骤三：配置插件参数

打开 ~/.openclaw/openclaw.json，在plugins部分添加配置：

```json
{
  "plugins": {
    "slots": {
      "memory": "modelstudio-memory-for-openclaw"
    },
    "entries": {
      "modelstudio-memory-for-openclaw": {
        "enabled": true,
        "config": {
          "apiKey": "sk-xxx",
          "userId": "user_001",
          "profileSchema": "your_profile_schema_id",
          "memoryLibraryId": "your_memory_library_id",
          "projectId": "your_project_id"
        }
      }
    }
  }
}
```
