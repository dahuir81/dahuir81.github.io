# DeepSeek

**类型**: AI公司/产品  
**领域**: 大语言模型、AI基础设施、光学压缩技术  
**创始人**: 梁文锋  
**总部**: 浙江省杭州市拱墅区  
**母公司**: 幻方量化

## 简介
DeepSeek（深度求索）是中国领先的AI大模型公司，由幻方量化于2023年7月创立。以技术开源和普惠AI著称。2025年初凭借DeepSeek-V3和R1模型爆火，以极高的性价比挑战OpenAI等巨头。2026年4月即将发布V4，迈入万亿参数时代。

## 核心产品

### 模型系列
- **DeepSeek V3** - 开源大模型，性能接近GPT-4
- **DeepSeek R1** - 推理模型，数学和代码能力强
- **DeepSeek V3.2** - 2025年12月发布，含Speciale版本
- **DeepSeek V4** - 下一代模型（2026年4月下旬发布）
  - 万亿参数 MoE 架构
  - 百万级 token 上下文窗口
  - Apache 2.0 开源，每百万 token 仅 0.3 美元
  - 首次适配华为昇腾等国产AI芯片
- **DeepSeek V4 Lite** - 轻量版，快速模式底层

### 核心技术
- **DeepSeek-OCR** - 光学压缩技术（2025年10月发布）
  - 论文: [arXiv:2510.18234](https://arxiv.org/abs/2510.18234)
  - 用视觉token代替文本token，压缩比10-20倍
  - 10×压缩下OCR精度>96%
  - 单张A100-40G每天处理3300万页
  - 可能是V4百万上下文的关键技术底座
- **DeepSeek-OCR2** - 2026年1月27日发布

### 产品形态
- **网页端** - chat.deepseek.com
  - 快速模式（V4 Lite）- 日常对话，支持多模态
  - 专家模式（V4）- 深度推理
  - Vision模式（即将上线）- 多模态视觉
- **API** - 开发者接口
- **iOS/Android App** - 移动端应用

## 商业模式演变

### 2025年：反商业阶段
- API定价极低
- 网页端完全免费
- 无功能分层

### 2026年：产品分层
- **快速模式** (V4 Lite) - 日常对话
- **专家模式** (V4) - 深度推理
- **Vision模式** (即将上线) - 多模态视觉

## 技术路线
- **MoE架构** - 混合专家模型，V4达万亿参数
- **开源策略** - 模型权重Apache 2.0开源
- **成本优化** - 极致推理成本控制（比GPT旗舰便宜10倍）
- **光学压缩** - DeepSeek-OCR，视觉token压缩10-20倍
- **国产芯片适配** - 首次支持华为昇腾，"去CUDA化"
- **长期记忆** - Engram条件化记忆架构，O(1)知识检索

## V4 四大技术突破
1. **长期记忆（LTM/Engram）** - 知识存储与推理层物理解耦
2. **编程能力跃升** - HumanEval 87.6%，SWE-Bench 83.7%
3. **原生多模态融合** - 文本/图像/音频/视频跨模态
4. **AI搜索强化** - 深度优化RAG能力
