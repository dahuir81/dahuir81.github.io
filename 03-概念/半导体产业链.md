# 半导体产业链

## 核心定义

半导体产业链从上游到下游的完整供应链体系，涵盖设计、制造、封装、设备、材料等环节。

## 产业链结构

| 环节 | 代表企业 |
|------|----------|
| **EDA/IP** | [[Synopsys]]、[[Cadence]] |
| **设备** | ASML、应用材料 |
| **材料** | 日本电子材料厂商 |
| **设计** | [[NVIDIA]]、[[AMD]]、[[华为昇腾]] |
| **制造（Foundry）** | [[台积电 TSMC]]、[[中芯国际 SMIC]] |
| **封装** | [[CoWoS]]、[[长电科技]]、[[盛合晶微]] |
| **存储** | [[SK海力士]]、[[三星]]、[[美光]] |
