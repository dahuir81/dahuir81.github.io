# Agent（智能体）

**定义**: 能自主感知、决策、行动的AI系统  
**核心能力**: 工具使用、任务规划、自主执行

---

## 关键特征

| 特征 | 说明 |
|------|------|
| **自主性** | 无需人工干预 |
| **工具使用** | 调用外部API、代码执行 |
| **规划能力** | 分解任务、制定策略 |
| **记忆** | 上下文学习、长期记忆 |

---

## 架构演进

1. **单Agent** - 独立任务
2. **多Agent** - 协作系统
3. **Agent框架** - OpenClaw、Claude Code等

## 价值转叙（2026-04）

**李强（腾讯云）观点**:
- "龙虾"热潮带来重要分水岭：AI从"对话"变成"可执行"
- 执行这一侧更容易让传统企业看到AI的真实价值
- B端用户意愿被极大提振，不同行业客户成为AI增长新来源

## 产业阶段转折（2026-04）

**从"模型军备竞赛"到"商业模式分化"的标志性事件**

2026年4月，中国两大科技巨头对 Agent/智能体的商业化路径出现明显分化：

- **阿里巴巴**：成立[[02-实体/Alibaba Token Hub|ATH事业群]]，将 Token 上升为集团级战略，走"基础设施扩张"路线
- **腾讯**：李强提出"Token=油耗、大模型=引擎"，走"模型+应用差异化"路线

这种分化意味着：
1. Agent 热潮已经从"技术追逐"进入"商业落地"阶段
2. 不同厂商开始基于自身基因选择不同的利润池
3. 纯卖 Token 的赚钱效率下降，应用层闭环的价值更加凸显
