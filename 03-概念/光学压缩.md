## 核心定义

光学压缩（Optical Compression）是一种利用视觉 Token 代替文本 Token 来压缩信息密度的技术。由 DeepSeek 团队（魏浩然、孙耀峰、李雨坤）于 2025 年 10 月提出，通过将文本内容编码为紧凑的视觉表示，可实现 10~20 倍的压缩率，有效缓解长上下文场景下的 KV Cache 压力和上下文窗口限制。

## 关键特征

- **高压缩率**：10× 压缩比内 OCR 精度 >96%，20× 压缩仍有约 60% 精度
- **视觉编码**：将文档图像通过 DeepEncoder（SAM + 卷积压缩器 + CLIP）编码为少量视觉 Token
- **KV Cache 节省**：更少的 Token 数量直接降低 KV Cache 的显存占用
- **算力节省**：注意力计算 O(n²)，Token 减少 10 倍则计算量减少约 100 倍
- **两段式架构**：DeepEncoder 编码压缩 → DeepSeek 3B MoE 解码还原文字
- **与长上下文协同**：100 万文本 Token 可压缩至约 5~10 万视觉 Token，使百万上下文处理成为可能
- **信息密度优势**：二维排版（位置、字号、框线）天然携带语义，信息密度远超纯文本
