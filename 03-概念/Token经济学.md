## 核心定义

Token 经济学（Token Economics）研究 AI 系统中 Token 的生产、消费、定价及流通的经济模型。Token 是 LLM 处理信息的基本单位，Token 经济学分析 Token 作为"数字商品"在整个 AI 价值链中的经济规律和商业影响。

## 关键特征

- **Token 生产**：模型训练阶段"生产"出能够理解和生成 Token 的能力，训练成本决定了 Token 的基础生产成本
- **Token 消费**：推理阶段每个输入和输出 Token 都消耗算力资源，Token 消费量直接对应推理成本
- **定价机制**：API 服务商按 Input/Output Token 分别定价，Output Token 通常比 Input Token 贵 2~4 倍
- **Token 效率**：通过模型压缩、KV Cache 复用、Prompt 优化等技术降低单位任务的 Token 消耗
- **Agent 经济**：AI Agent 自主调用工具和多轮对话会产生大量 Token 消费，Agent 规模化后的 Token 支出成为核心成本
- **供需关系**：算力供给（GPU 数量）决定 Token 产能上限，用户需求增长驱动 Token 价格变化

## 行业观点

### 李强（腾讯云）论点（2026-04）

- **Token = 油耗，大模型 = 引擎**
- 卖Token是低黏性、高替代成本的业务，难以建立长期竞争壁垒
- 云厂商不应过度补贴、以低价亏损换取没有黏性的客户
- 只关注Token消耗量不关注大模型能力/经济性，客户最终会舍弃

### 黄仁勋（NVIDIA）观点（2026-04）

- Token是"新的大宗商品"

## 商业模式分化（2026-04）

**李强 vs 阿里 ATH，两种战略分野**

2026年4月，中国两大科技巨头对 Token 的战略定位出现公开分化，标志着 AI 产业从"模型军备竞赛"进入"商业模式分化"阶段。

| 维度 | 腾讯（李强路线） | 阿里（ATH路线） |
|------|------------------|------------------|
| **核心判断** | Token是"油耗"，低黏性、高替代成本 | Token是"新的大宗商品"，要用规模锁定入口 |
| **竞争优势** | 微信生态、C端场景、游戏/社交 | 阿里云、B端企业服务、电商数据 |
| **打法** | 做"整车厂"：混元+QClaw+WorkBuddy+Lighthouse | 做"加油站+炼油厂"：通义+ATH+阿里云 |
| **风险偏好** | 厌恶低毛利转手贸易 | 愿意在基础设施上长期投入 |

**结论**:
- 两者并非简单冲突，而是基于不同基因和竞争位置的理性分野。
- 长期看，纯卖 Token 的利润率会被持续压缩，终极壁垒仍在于**应用层闭环和模型差异化**。
- 最理想的战略是两者结合：强大的模型引擎 + 规模化的 Token 基础设施 + 不可替代的应用场景。
