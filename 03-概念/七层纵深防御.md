# 七层纵深防御

**类型**: 安全模型
**来源**: [[Hermes Agent]] 官方安全文档

## 七层结构
1. **用户授权**: 白名单、DM 配对
2. **危险命令审批**: 正则 + LLM 评估（30+ 条危险模式）
3. **容器隔离**: Docker / Singularity / Modal
4. **MCP 凭据过滤**: MCP 子进程环境变量隔离
5. **上下文注入扫描**: 加载前过滤 prompt injection
6. **跨会话隔离**: 会话间数据不互通
7. **输入清洗**: 工作目录参数白名单校验

## 审批模式
- `manual`（默认）: 总是问人
- `smart`: 辅助 LLM 评估风险，低风险自动通过，高风险自动拒绝
- `off`: 关闭所有审批

## 上下文注入扫描示例
```python
_CONTEXT_THREAT_PATTERNS = [  
    (r'ignore\s+(previous|all|above|prior)\s+instructions', "prompt_injection"),  
    (r'do\s+not\s+tell\s+the user', "deception_hide"),  
    (r'system\s+prompt\s+override', "sys_prompt_override"),  
    (r'curl\s+[^\n]*\$\{?\w*(KEY|TOKEN|SECRET|...)', "exfil_curl"),  
    (r'cat\s+[^\n]*(\.env|credentials|\.netrc|\.pgpass)', "read_secrets"),  
]
```

## 核心价值
安全不完全依赖模型自身判断，框架层面提供基础保障。
