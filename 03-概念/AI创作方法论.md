# AI 创作方法论

> **类型**：Concept
> **创建**：2026-04-09
> **来源**：白鸽《脱口秀 Skill 与神之一手》、卡兹克开源 Skill

## 定义

用 AI 辅助内容创作的系统方法，核心是：让 AI 理解创作者的逻辑和边界，而不是模仿表面风格。

## Skill 迭代法（卡兹克方案）

```
代表性文章(2-3篇) → AI总结Skill → AI写稿 → 人修改
→ 差异分析 → 迭代回Skill → 重复3-4轮
```

### 四层自检

1. 硬性规则扫描
2. 风格一致性检查
3. 内容质量审查
4. **活人感终审**："是人写的还是AI输出的？"

## 好 Skill 的五个要素

| 要素 | 说明 |
|------|------|
| 方法论 | 创作逻辑，不是风格规则 |
| 踩过的坑 | 失败教训，不只是成功经验 |
| 边界感 | 你绝对不会说什么话 |
| 好恶 | 你喜欢什么、讨厌什么 |
| 节奏控制 | 信息密度和呼吸感 |

## 核心判断

- AI 是**加速器**，不是创造者
- 迭代 3-4 轮足够，过拟合会限制创新
- 目标是构建**自己**，不是复制**别人**
- 灵魂 = 经历 + 情绪 + 跨领域概念嫁接
