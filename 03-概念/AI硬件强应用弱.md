# AI硬件强应用弱

**类型**: 市场趋势 / 投资主题
**来源**: [[2026-04-09-科技早报-source]]

## 核心现象
AI产业链呈现**硬件强、应用弱的极致分化格局**。

## 数据表现
| 指标 | 数值 |
|------|------|
| IGV与SOX单日收益差 | -727个基点（创五年最差） |
| 年内半导体 vs 软件收益差距 | 扩大至35% |
| 半导体52周新高 | 多家公司逼近 |
| 软件52周低位 | 超10家企业处于附近 |

## 驱动因素
**硬件强**：
- AI算力需求真实且持续加速（台积电Q1毛利率67%）
- 计算能力仍是市场最紧缺资源
- 半导体、存储、光模块全线大涨

**应用弱**：
- 模型迭代加速导致软件应用层遭遇资金挤出
- 资金只认"卖铲子"逻辑
- 应用端缺乏杀手级产品

## 核心观点
高盛坚定押注AI "picks and shovels"（卖铲子）赛道：
- AI基础设施需求未受地缘影响
- AI资本支出将持续加速且具备长期韧性
- 行业处于早中期周期，AI芯片生态系统仍有充足增长空间
