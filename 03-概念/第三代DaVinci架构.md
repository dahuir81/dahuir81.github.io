## 概述

第三代DaVinci架构（DaVinciCore）是华为昇腾AI芯片的核心计算架构，在第二代基础上针对Transformer、推荐、多模态等场景进行了全面升级。

---

## 关键改进

### 1. 算力效率革命

| 改进点 | 说明 |
|--------|------|
| **低精度计算** | 新增HiF8/FP8/MXFP8/MXFP4支持 |
| **Cube-Vector协同** | 新增CV融合通路，解决混合计算瓶颈 |
| **Vector翻倍** | Vector核FP16/FP32性能翻倍 |
| **Cube:Vector配比** | 8:1 |
| **L0C buffer** | 增加到256KB，支持256*256 tile |
| **算力密度** | 提升20%，90%场景覆盖面 |

### 2. HiF8动态浮点格式

- **编码**: 变长前缀编码 + 原码阶码优化
- **动态范围**: 接近FP16（-22~15）
- **计算效率**: 较FP8提升2~4倍
- **适用场景**: LLM训练与推理

### 3. 访存效率优化

| 指标 | 上一代 | 新一代 | 提升 |
|------|--------|--------|------|
| L2 Cache粒度 | 512B | 128B | 4x（小包场景） |
| L2 Control Hint | - | 支持多种 | 新增 |
| ND2NZ | 基础 | Advance模式 | Stride≤256B支持并包 |

### 4. SIMD/SIMT混合编程

- **SIMD**: 规则计算，双发指令榨干性能
- **SIMT**: 复杂逻辑/多条件分支/小包搬运
- **优势**: 代码可读性与执行效率兼得

### 5. NDDMA高效数据访问

- **传统方式**: 硬件地址计算 + 访存合并（需多步）
- **新方式**: 一行NDDMA指令完成（transpose/stride/broadcast/slice）
- **效率提升**: 50%以上

### 6. 极简同步机制

- **旧机制**: set/wait配对，繁琐且易出错
- **新机制**: BufferID模型，消除分支逻辑
