# Alibaba Token Hub (ATH)

**定义**: 阿里巴巴于2026年3月16日成立的事业群，以"创造Token、输送Token、应用Token"为核心的统一调度中心  
**负责人**: [[吴泳铭]]（亲自挂帅）  
**战略地位**: 与电商、云智能并列，Token被提升为集团战略核心计量单位

## 背景

阿里深陷"流量去哪儿了"困境数年，需要找到一张AI时代或者关乎未来的门票。

## 核心目标

> "未来所有的商业行为都将以Token为计量单位结算"

阿里不满足于"AI帮电商多卖1%的货"，而是赌一个范式革命：
- **创造Token**：大模型研发与训练
- **输送Token**：AI云基础设施与推理平台
- **应用Token**：将Token能力嵌入所有业务线

## 风险

| 维度 | 问题 |
|------|------|
| **财务** | 2026Q3净利润-67%，自由现金流-71%；承诺3800亿AI投入 |
| **文化** | 从"运营为王"到"AI创新"的组织基因突变 |
| **用户** | 千问DAU曲线来也匆匆去也匆匆，红包退潮后明显下滑 |
| **时间** | 造AI底座是无底洞，数年内可能看不到财务回报 |

## 与阿里云的区别

阿里云是"从业务中长出来"的路径——被双11交易洪峰逼出来的算力自给自足。  
ATH是"技术驱动业务"的路径——AI成为凌驾于具体业务之上的"新底座"。

**逻辑反转**：过去是业务驱动技术，现在是技术定义业务。
