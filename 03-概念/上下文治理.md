# 上下文治理（Context Governance）

## 核心定义

上下文治理是AI Agent的上下文管理框架，涵盖对话上下文管理、压缩（compaction）、长期记忆、技能沉淀等核心能力。

## 关键特征

### 核心组件
| 组件 | 功能 | 实现 |
|------|------|------|
| **会话管理** | 跟踪对话历史 | Session日志 |
| **压缩机制** | 上下文压缩 | compact机制 |
| **长期记忆** | 跨会话持久化 | MEMORY.md + FTS5检索 |
| **技能沉淀** | 能力结构化存储 | 技能文件 |

### 在Claude Code中的实现
- Session日志：仅追加，重启不丢
- compact机制：自动压缩上下文
- 长期记忆系统：跨会话推理

### 在Hermes Agent中的实现
- MEMORY.md + USER.md：FTS5检索 + 大模型摘要
- 三层学习闭环：记忆 → 技能 → 训练数据
