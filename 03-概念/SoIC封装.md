# SoIC（System-on-Integrated-Chips）

**全称**: System-on-Integrated-Chips  
**中文**: 系统级集成芯片  
**类型**: 3D封装技术  
**厂商**: 台积电（TSMC）  
**推出时间**: 2020年

---

## 技术原理

SoIC是一种**3D封装技术**，通过**直接键合**（Direct Bonding）将多个芯片（Chiplet）在垂直方向上堆叠，形成三维集成。

### 与CoWoS的区别

| 维度 | CoWoS | SoIC |
|------|-------|------|
| **维度** | 2.5D | **3D** |
| **堆叠方式** | 芯片并排放在硅中介层上 | 芯片垂直堆叠 |
| **互连方式** | 通过硅中介层走线 | **直接铜-铜键合** |
| **互连密度** | 中等 | **极高** |
| **延迟** | 较低 | **极低** |
| **功耗** | 较低 | **更低** |
| **典型应用** | GPU+HBM | CPU+缓存、逻辑+逻辑 |

---

## SoIC的两种技术路线

### 1. SoIC-P（SoIC with Polymer）

**技术特点**:
- 使用聚合物粘合剂
- 适合较大间距的互连
- 成本相对较低

**应用场景**:
- 存储器堆叠
- 传感器集成

### 2. SoIC-C（SoIC with Copper）

**技术特点**:
- **铜-铜直接键合**（Cu-Cu Bonding）
- 互连间距可小至**9微米**
- 极高的互连密度和带宽
- 极低的信号延迟

**应用场景**:
- 高性能计算（HPC）
- AI加速器
- CPU+SRAM堆叠

## SoIC vs CoWoS 技术对比

### 结构对比

```
CoWoS (2.5D):
┌─────────────────────────────────┐
│  Chip1    Chip2    Chip3 (HBM)  │
│   ↓         ↓         ↓         │
│   └─────────┴─────────┘         │
│      Silicon Interposer         │
│              ↓                  │
│         Substrate               │
└─────────────────────────────────┘

SoIC (3D):
┌─────────────────┐
│    Top Chip     │  ← 可能是SRAM/缓存
│   ↓↓↓↓↓↓↓↓↓    │
│   Cu-Cu Bonding │  ← 直接铜键合
│   ↑↑↑↑↑↑↑↑↑    │
│   Bottom Chip   │  ← 可能是CPU/GPU
│         ↓       │
│    Substrate    │
└─────────────────┘
```

### 性能对比

| 指标 | CoWoS | SoIC | 提升 |
|------|-------|------|------|
| **互连间距** | ~55微米 | **~9微米** | 6倍 |
| **互连密度** | ~10K/mm² | **~100K/mm²** | 10倍 |
| **带宽密度** | 高 | **极高** | 5-10倍 |
| **延迟** | 低 | **极低** | 50%↓ |
| **功耗** | 低 | **更低** | 30%↓ |

## 技术优势

### 1. 性能提升
- **带宽**: 互连密度提升10倍，带宽大幅提升
- **延迟**: 垂直互连比水平走线延迟更低
- **功耗**: 短距离互连降低功耗

### 2. 灵活性
- **异构集成**: 不同工艺节点的芯片可以堆叠
- **模块化**: CPU、GPU、SRAM、I/O可以分别优化后集成

### 3. 成本效益
- **良率优化**: 大芯片拆分成小芯片（Chiplet），良率提升
- **混合工艺**: 高性能部分用先进制程，I/O部分用成熟制程

---

## 技术挑战

### 1. 散热
- 3D堆叠导致散热困难
- 需要先进的散热方案（如液冷）

### 2. 设计复杂度
- 3D设计工具链不成熟
- 热管理和信号完整性分析复杂

### 3. 测试难度
- 堆叠后难以测试底层芯片
- 需要先进的可测试性设计（DFT）

### 4. 产能瓶颈
- SoIC产能远低于CoWoS
- 设备投资大，扩产周期长

## 与CoWoS的关系

### 互补而非替代

```
应用场景选择:

需要HBM高带宽 → CoWoS (2.5D)
     ↓
需要逻辑+缓存堆叠 → SoIC (3D)
     ↓
两者都需要 → CoWoS + SoIC 组合
```

### 组合应用案例

**Google TPU v9（2028年）**:
- **SoIC**: 3D堆叠计算die和I/O die
- **CoWoS-L**: 连接SoIC堆叠体和HBM

**NVIDIA Feynman高端版（2028年）**:
- **SoIC**: GPU die + SRAM die 3D堆叠
- **CoPoS/CoWoS**: 封装SoIC堆叠体和HBM

## 核心结论

1. **SoIC是3D封装技术**，与CoWoS（2.5D）互补
2. **铜-铜直接键合**实现极高互连密度（~9微米）
3. **2027-2028年需求爆发**：TPU v9、Trainium 4、Feynman等采用
4. **产能快速扩张**：2028年达65K wfpm（较2027年翻倍）
5. **技术挑战**：散热、设计复杂度、测试难度仍需突破
