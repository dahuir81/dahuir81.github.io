# CoWoS（Chip-on-Wafer-on-Substrate）

**全称**: Chip-on-Wafer-on-Substrate
**类型**: 2.5D先进封装技术
**发明者**: 台积电（TSMC）
**推出时间**: 2011年

## 技术原理

CoWoS是一种2.5D封装技术，将多个芯片（Chip）集成在一个硅中介层（Silicon Interposer）上，再通过硅中介层连接到封装基板（Substrate）。

### 结构层次
1. **顶层**: 计算芯片（GPU/CPU/AI加速器）
2. **中层**: 硅中介层（提供高密度的芯片间互连）
3. **底层**: 封装基板（连接到PCB）
4. **HBM**: 高带宽内存（通常位于计算芯片旁边）

## 技术优势

### 性能优势
- **高带宽**: 芯片间互连带宽远高于传统封装
- **低延迟**: 芯片间通信延迟显著降低
- **高集成度**: 可在同一封装内集成多个芯片和HBM

### 应用场景
- AI训练芯片（NVIDIA A100/H100/B200）
- 高性能计算（HPC）
- 数据中心芯片
- 网络处理器

## 产业链

### 设备与材料
- **硅中介层**: 台积电自产
- **封装设备**: ASMPT、K&S等
- **基板**: 揖斐电、欣兴电子等
- **HBM**: 海力士、三星、美光

## 中国CoWoS发展

### 现状
- 主要依赖台积电、日月光
- 国内长电科技、通富微电布局中
- 受出口管制影响，先进封装设备受限

### 挑战
- 硅中介层制造技术
- 高精度封装设备
- HBM供应
