# 开源 (Open Source)

**定义**: 模型权重、代码公开可获取  
**代表**: Llama, DeepSeek, Mistral  
**vs 闭源**: GPT-4, Claude

---

## 开源优势

| 优势 | 说明 |
|------|------|
| **可定制** | 微调、蒸馏 |
| **透明** | 可审计、可研究 |
| **低成本** | 无需API费用 |
| **生态** | 社区驱动创新 |

---

## 开源风险

- **滥用** - 恶意微调
- **安全** - 移除安全限制
- **竞争** - 削弱闭源厂商
