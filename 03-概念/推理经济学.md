# 推理经济学（Inference Economics）

**定义**: 研究AI推理（inference）场景下的成本效率、性能权衡和商业可行性的分析框架

## 核心概念

在大模型时代，**推理**（使用已训练模型生成输出）的成本和效率比**训练**（训练模型本身）更影响商业可行性。推理经济学关注的是"每token成本"而非"峰值算力"。

## 关键指标

### 1. TCO（总拥有成本）

$$TCO = 芯片采购成本 + 电力成本 + 运维成本 + 软件适配成本$$

**Morgan Stanley发现**:
- 国产芯片TCO显著低于NVIDIA中国特供版（H200等）
- 主要原因：芯片采购价格更低、电力成本优势

### 2. 每Token成本

$$每Token成本 = TCO / 总推理Token数$$

**关键洞察**:
- 国产芯片每token成本与NVIDIA处理器**相当**
- 尽管峰值算力落后，但因价格更低，单位经济性不输

### 3. TPS（每秒Token输出）

$$TPS = f(算力, 内存带宽, 模型大小, 批处理大小)$$

**Morgan Stanley分析假设**:
- 模型: DeepSeek R1
- 输入Token: 1,024
- 输出Token: 1,024
- 活跃专家数: 9/257
- 模型大小: 671GB
- 计算精度: FP8（如无则FP16）

### 4. 每瓦性能（Performance per Watt）

$$性能/瓦 = TPS / 功耗$$

**Morgan Stanley发现**:
- 领先国产加速器每瓦性能与NVIDIA A100**持平**
- GB300每瓦token性能可达H200的**50倍**（DeepSeek R1，但无法出口中国）

### 为什么峰值算力不是关键？

1. **推理场景**: 大多数商用AI是推理而非训练
2. **成本敏感**: 推理需要大规模部署，总成本是关键
3. **软件优化**: 好的编译器/框架可以弥补硬件差距
4. **集群扩展**: 通过更多芯片互联弥补单卡差距

## 中国AI数据中心的TCO优势

1. **芯片成本**: 国产芯片价格远低于NVIDIA旗舰产品
2. **电力成本**: 中国数据中心电价较低
3. **规模效应**: 互联网大厂AI capex快速增长
4. **政策支持**: 地方政府补贴和优惠政策

## 竞争格局演变

### 短期（2026-2027）
- DeepSeek等开源模型降低推理门槛
- 国产芯片在推理场景已具经济性
- NVIDIA H200等中国特供版仍有一定市场

### 中期（2028-2030）
- 国产芯片产能爬坡，供应更充足
- 76%自给率目标
- 软件生态逐步完善

### 长期
- 性能差距缩小，成本优势扩大
- 推理经济学成为核心竞争力

## 与训练经济学的对比

| 维度 | 训练经济学 | 推理经济学 |
|------|-----------|-----------|
| 核心指标 | 训练时间、收敛速度 | 每token成本、延迟 |
| 硬件需求 | 极致算力、大显存 | 性价比、能效比 |
| 规模 | 千卡/万卡集群 | 单卡到百卡 |
| 成本敏感 | 中（一次性投入） | 高（持续运营） |
| 国产芯片竞争力 | 较弱 | **较强** |
