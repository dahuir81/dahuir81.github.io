# Skill系统

## 核心定义

Skill系统是AI Agent的能力框架，将Agent的可复用能力结构化为可调用、可组合、可沉淀的技能单元。

## 关键特征

### Skill来源
| 来源 | 说明 |
|------|------|
| **内置** | Agent自带核心能力 |
| **社区** | 用户贡献共享 |
| **自动生成** | 完成任务后沉淀 |
| **训练** | 强化学习生成 |

### 与Claude Skill的关系
- [[Claude Skill]]: Anthropic的官方技能系统
- Skill系统: 通用的Agent能力框架
- 两者互补，Claude Skill是Skill系统的具体实现
