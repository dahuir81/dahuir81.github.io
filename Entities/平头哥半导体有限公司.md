# 平头哥半导体有限公司 (T-Head Semiconductor)

## 基本信息

| 项目 | 内容 |
|------|------|
| **公司全称** | 平头哥半导体有限公司 |
| **英文名称** | T-Head Semiconductor Co., Ltd. |
| **成立时间** | 2018年9月 |
| **母公司** | 阿里巴巴集团 |
| **前身** | 杭州中天微系统有限公司（2018年被阿里收购） |
| **总部地点** | 中国杭州 |
| **公司类型** | 有限责任公司（非上市） |

## 发展历程

### 2018年：成立
- 阿里巴巴收购中天微系统，组建平头哥半导体
- 孟建熠（原中天微副总经理）加入，负责RISC-V业务

### 2019-2023年：RISC-V布局
- 发布玄铁系列RISC-V处理器
- 推动RISC-V与安卓、Linux等操作系统适配
- 成为RISC-V国际基金会核心成员

### 2024-2025年：AI芯片突破
- 2026年1月：真武810E AI芯片正式发布
- 400+客户验证，多个万卡集群部署
- 镇岳510 SSD主控规模化商用

## 核心团队

### 现任管理层
- **负责人**: 阿里巴巴达摩院/云智能事业群统筹

### 关键人物变动
- **孟建熠离职** (2025年): 原副总裁、RISC-V业务灵魂人物离职，具体影响待观察

## 产品矩阵

### AI芯片
- **真武PPU系列**: 训练/推理加速芯片，对标英伟达H20

### 存储芯片
- **镇岳510**: 企业级SSD主控芯片

### RISC-V处理器
- **玄铁C920**: 高性能应用处理器
- **玄铁C907**: AIoT/边缘计算处理器

## 战略定位

### "通云哥"黄金三角
平头哥与阿里云、通义实验室形成阿里AI基础设施的三大支柱：
- **芯片架构**（平头哥）
- **云平台架构**（阿里云）
- **模型架构**（通义实验室）

### 全球地位
- 与谷歌并列：全球唯二具备"大模型+云+芯片"全栈能力的公司
- RISC-V领域：中国投入最大、国际社区贡献最多的机构

## 财务与商业化

### 核心数据（阿里2026财年Q3财报）
- **累计交付**: 47万片GPU芯片（截至2026年2月）
- **年化收入**: 已达百亿级别（吴泳铭披露）
- **外部客户占比**: 60%（28万片服务外部客户）
- **客户数量**: 400+家

### 阿里集团投资
- **未来三年**: 3800亿元（云+AI基础设施）
- **2025年资本支出**: 1100-1200亿元
- **服务器采购**: 700-800亿元

### 商业化进展
- 真武芯片：供不应求，2025年3月涨价34%
- 已成为**国产GPU出货量第一**
