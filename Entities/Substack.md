# Substack

**类型**: 内容订阅平台  
**定位**: 知识和信息平权，为独立研究提供商业平台

## 核心服务

### 1. 支付与订阅闭环
- 深度整合Stripe支付
- 一键开启按月或按年付费订阅
- 平台不干涉定价（Citrini年费数千美金）
- 直接筛选高净值客户

### 2. 邮件直达
- 与算法推荐不同，核心是邮件驱动
- 研究报告直接发到读者邮箱
- "强制送达"确保专业洞察不被信息流淹没

### 3. 私域流量所有权
- 订阅者邮箱名单完全属于作者
- 即使离开Substack也可带走全部用户

### 4. AI辅助与精简排版
- 干净、无广告的阅读体验
- 集成AI工具帮助数据处理和排版校对
- 10人以下小团队可做出比肩大投行的精美研报

## 流量来源

- X (Twitter)
- 其他外部链接
- 口碑传播

## 对比中文平台

与微信公众号、知乎等中文知识信息平台相比，Substack的特点：
- 更强调付费订阅和私域流量
- 邮件驱动而非算法推荐
- 作者拥有完全的订阅者数据
