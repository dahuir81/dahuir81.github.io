# Project Glasswing

**类型**: 项目/组织  
**发起方**: Anthropic  
**成立时间**: 2026年4月

## 简介
Project Glasswing 是 Anthropic 牵头成立的安全项目，旨在将 Mythos 工具优先提供给防御方（关键基础设施和开源项目），让他们在类似能力的模型广泛可用之前修复最重要的漏洞。

## 背景
- Claude Mythos 发现了大量「零日漏洞」
- 这些漏洞存在于 OpenBSD、FFmpeg、FreeBSD 等核心系统
- 过去安全是因为「攻击需要大量人力」
- 现在「几小时就能完成同样工作」

## 目标
- 优先给防御方（关键基础设施、开源项目）
- 在攻击者之前修复漏洞
- 在类似能力的模型广泛可用之前完成

## 策略
- 负责任披露 vs 威胁示警的平衡
- 公开展示危险性，建立安全权威地位
- 向潜在客户证明 Mythos 能力上限
