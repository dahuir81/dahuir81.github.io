# Nous Research

**类型**: 公司 / 研究机构
**项目**: [[Hermes Agent]] (开源 Agent Runtime)
**GitHub**: https://github.com/NousResearch/hermes-agent

## 背景
Nous Research 是一家 AI 研究机构，以 Hermes 系列模型闻名，后将积累应用于 Agent 系统开发。

## 产品矩阵
- **HERMES 系列大模型**: HERMES-4 达到 405B 规模
- **Psyche**: 模型分布式训练基建
- **Hermes AGENT**: 开源 Agent Runtime
- **API Portal**: 基于订阅的 LLM 服务，零配置
- **整体定位**: 颇有点"小众版 DeepSeek"的即视感

## Hermes Agent 工程质量
- 约 3000 个测试
- WAL 模式 SQLite + FTS5 全文检索
- prompt caching 保护
- 六种终端后端支持
- 七层安全防御
- 122 个内置 SKILL.md
