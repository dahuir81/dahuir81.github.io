## 一、组织架构调整（2026-04-08）

### 事件

4月8日，阿里巴巴集团CEO吴泳铭发布内部信，宣布AI相关组织调整：

1. **设立集团技术委员会**
   - 组长：吴泳铭
   - 成员：周靖人、吴泽明、李飞飞

2. **通义实验室升格为通义大模型事业部**
   - 从"支撑部门"变为"战略核心单元"
   - 直接扛AI营收指标

3. **人事变动：三权分立**

| 人物 | 新角色 | 负责范围 |
|------|--------|----------|
| 周靖人 | 首席AI架构师 + 通义事业部负责人 | AI模型和产品（通义/Wan系列） |
| 李飞飞 | 阿里云CTO | 云技术及AI云基础设施建设 |
| 吴泽明 | 集团业务技术平台 | 集团技术底座 + AI推理平台 |

### 关键信号

- 通义从"实验室"升格为"事业部"，从后台变前台
- 技术委员会是跨部门的决策层，不是执行层
- 吴泳铭亲自担组长，说明AI是一号工程

## 二、AI营收目标

吴泳铭提出：**未来五年AI与云相关业务年营收突破1000亿美元。**

核心判断："当越来越多的企业开始在内部启用由大模型驱动的Agent来完成端到端的工作任务时，整个AI和云面向的IT预算市场发生了根本性的变化。"

关键洞察：企业消耗Token不再被视为IT预算，而是视为生产成本或研发成本。

## 三、HappyHorse-1.0 视频模型

### 基本信息

| 属性 | 详情 |
|------|------|
| 名称 | HappyHorse-1.0（欢乐马） |
| 发布方式 | 匿名发布（The Information率先报道） |
| 排名 | Artificial Analysis全球第一，击败Seedance 2.0 |
| 类别 | 文字到视频、图片到视频 双冠军 |
| 技术架构 | 40层单流Transformer + 8步去噪推理 |
| 开源 | 官方确认即将开源（GitHub + 权重） |
| 在线体验 | happyhorse.app 免费试用 |

### 背后团队

**张迪**：
- 上海交通大学本硕，阿里P11
- 2010年加入阿里
- 2020-2025：快手副总裁，可灵AI从0到1的核心推手（"可灵之父"）
- 2025年9月：短暂加入B站，任技术条线负责人
- 2025年11月：回归阿里，组建淘天未来生活实验室
- 仅用5个月打造HappyHorse-1.0

**汇报线**：张迪 → 郑波（淘天首席科学家/阿里妈妈CTO）

### 与Wan系列的关系

技术层面高度同源：

| 维度 | Wan 2.x | HappyHorse |
|------|---------|-----------|
| 架构 | 单一DiT Transformer | 40层单流Transformer |
| 多模态 | 全模态输入统一处理 | 文本+图像+视频+音频联合去噪 |
| 推理 | 8步去噪 | 8步去噪（完全一致） |

结论：HappyHorse极有可能是Wan系列的重大升级版，但团队独立于通义事业部。

### 赛马制架构

```
吴泳铭（集团CEO+技术委员会组长）
├── 周靖人（首席AI架构师）→ 通义事业部 → Wan 系列
└── 郑波（淘天首席科学家）→ 张迪（未来生活实验室）→ HappyHorse
```

## 四、匿名发布的四个动机

1. **打快手的脸**：可灵之父带着新模型把可灵和Seedance都踩在脚下，证明核心技术在人不在公司
2. **绕过组织惯性**：独立运作先打一波，出成绩再收编，阿里赛马逻辑
3. **悬念营销**：中国AI圈流行先匿名后揭晓，小米上个月用Pony Alpha干过
4. **资本市场叙事**：为1000亿美元AI营收目标提供多线支撑，视频生成是最性感的应用场景

## 六、张迪放淘天的深层动机

1. **贴近应用场景**：淘天有电商直播、商品视频、商家内容生产，视频生成直接可用
2. **避开通义惯性**：通义刚升格，内部消化中，独立出来反而快跑
3. **蒋凡的野心**：淘天CEO要证明自己不只管电商，AI视频是突破边界的武器
4. **挖人的诚意**：独立实验室+直接汇报，比塞进通义当中层有吸引力

## 七、商业判断

1. **技术能力**：阿里在AI视频领域已做到全球第一，但缺生态（没有抖音/快手这样的内容平台）
2. **组织风险**：赛马制效率高，但两条线最终怎么整合？Wan和HappyHorse会打架
3. **估值影响**：港股两天涨15%，但从52周高点看还是打了六六折
4. **百炼平台**：阿里云已准备将HappyHorse开放给企业客户，大概率进入百炼API
