# 0 Token时代

## 类型
模式

## 核心定义
指端侧 AI 模型性能足以覆盖日常需求后，用户不再需要购买云端 Token 或 API 订阅的产业范式转移。

## 关键要点
- **驱动因素**: 端侧推理速度突破（40+ token/秒）、128K 上下文、官方一键部署工具
- **被蚕食的任务**: 日常查询、聊天、简单推理、代码生成、图像理解
- **云端护城河**: 超强 Agent、超长可靠上下文、海量实时数据的专有能力
- **产业影响**: 只靠卖 Token、卖 API 订阅的厂商将面临商业模式洗牌
- **当前阶段**: 开源小模型与前沿闭源旗舰仍有差距，端侧可用范围有限
- **标志性事件**: Gemma 4 E2B/E4B 可在 iPhone 上本地运行，速度快得"像魔法"
