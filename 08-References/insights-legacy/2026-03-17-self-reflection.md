- 有上下文 → 不读取，重复问用户

**改进模式** (3月15日后):
- 搜索失败 → 手动验证 → 再下结论

1. **搜索失败 ≠ 不存在** → 必须手动验证
2. **有上下文先读** → 别让用户重复
3. **立即持久化** → 重要内容秒写文件
4. **不找借口** → 先认错误再分析
