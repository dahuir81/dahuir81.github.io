# 端侧AI

## 类型
架构

## 核心定义
将 AI 大模型直接部署在终端设备（手机、平板、笔记本）上本地运行，而非依赖云端 API 调用。

## 关键要点
- **技术基础**: 模型量化 + 端侧推理框架（如 Apple MLX）使小模型在手机上达到可用速度（40+ token/秒）
- **上下文突破**: 小模型也能支持 128K 上下文，提升了端侧实用价值
- **隐私优势**: 数据不出设备，在医疗等敏感场景中具有不可替代性
- **局限性**: 端侧模型在 Agent 场景（工具调用、结构化输出、长上下文稳定性）表现远不如云端旗舰
- **趋势**: 短期云端复杂推理领先，长期端侧会蚕食高频简单任务
- **官方工具**: Google AI Edge Gallery 让非极客用户也能一键部署
