# Claw AI Lab

**类型**: 多Agent协同科研系统  
**创始人**: [[Liu Fayao]], [[Ye Deheng]], [[Chen Tianrun]]  
**核心思想**: 让研究从"单打独斗"变成"带一个实验室"

---

## 核心痛点

传统科研：
- 一个人方向偏了没人提醒
- 遇到歧义没人讨论
- 结果不对只能反复试错

**Claw方案**：像实验室一样，不同角色同时推进，不同路径并行展开。

---

## 金字塔式分层架构

```
研究方向设定 (PI)
    ↓
方法设计与实验规划 (研究员)
    ↓
代码实现与结果分析 (工程师)
```

- 每层专属Agent负责
- 任务队列与上下文连接
- 上层决策根据下层结果动态调整

---

## 三种模式

| 模式 | 特点 |
|------|------|
| **Lab讨论模式** | 多方向并行，跨方向讨论达成共识 |
| **Lab独立研究模式** | 多方向并行，独立生成假设 |
| **论文复现模式** | 单Agent全流程复现 |

---

## Claw Code Harness

**核心创新**：建立从想法到实验结果的可信执行链路。

### Turn Loop
```
读取代码库/数据集
    ↓
理解 → 编写 → 测试 → 修复
    ↓
循环直到成功
```

### Experiment Harness
- 时间预算控制
- 指标上报
- 异常值校验
- 标准化results.json

---

## 群体智能

**示例**：具身智能实验室讨论
- VLM、VLA、World Model三个方向
- 分歧展开 → 假设暴露 → 证据对齐 → 路线重组
- 产出更强共识和清晰优先级
