# MCP (Model Context Protocol)

**全称**: Model Context Protocol  
**提出者**: Anthropic  
**目的**: 标准化AI与外部工具的交互

---

## 核心思想

统一协议，让AI能：
- 调用外部工具
- 访问本地文件
- 执行代码
- 与各种服务交互

---

## 争议

**支持方**:
- 标准化降低集成成本
- 生态统一

**质疑方**:
- 过于复杂
- 灵活性不足
- 大厂可能不跟进

---

## 与Skill对比

| 维度 | MCP | Skill |
|------|-----|-------|
| 层级 | 协议层 | 应用层 |
| 复杂度 | 高 | 低 |
| 灵活性 | 标准约束 | 自由定义 |
